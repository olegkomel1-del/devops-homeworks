
### Установка интерфейса командной строки Yandex Cloud (YC CLI)

```bash
curl https://storage.yandexcloud.net/yandexcloud-yc/install.sh | bash
```


Для управления облачной инфраструктурой скачал и запустил официальный инсталляционный скрипт с зеркала хранения Яндекса. После завершения установки обновил переменные окружения командой `source ~/.bashrc`.

> **Вывод команды yc --version:**
> ```text
> YC CLI v0.125.0 (или ваша версия, которая вывелась в терминале)
> linux/amd64
> ```
